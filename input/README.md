# Input

Hier landet angeliefertes Rohmaterial für die technische Pipeline.

## Unterordner
- `songs/` → Roh- oder Final-Audio als Input
- `artwork/` → Cover-/Artwork-Dateien
- `metadata/` → Song-Metadaten für die saubere Pipeline

## Hinweis
Für den einfachen aktuellen Übergabe-Workflow bleibt `upload/` parallel bestehen.
`input/` ist das Zielbild für den ausgebauten PoC.
