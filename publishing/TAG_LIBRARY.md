## Upload-Tags vs. Beschreibungs-Hashtags – Der Unterschied

| | Upload-Tags (Formular) | Beschreibungs-Hashtags |
|---|---|---|
| **Wo?** | Im Upload-Formular unter "Tags" | Am Ende der Videobeschreibung |
| **Sichtbar?** | Nein (nur für Algorithmus) | Ja (klickbar unter dem Titel) |
| **Anzahl** | 15-25 empfohlen | Max 3-5 (mehr = Spam-Wirkung) |
| **Zweck** | YouTube-Algorithmus versteht dein Video | Zuschauer finden dich über Hashtag-Suche |
| **Format** | Kommagetrennt, Leerzeichen erlaubt | #OhneLeerschritte |

## Update-Leitfaden

### Wann prüfen?
- **Alle 2-3 Monate:** YouTube Studio → Analytics → Traffic Sources → YouTube Search
- **Bei jedem GoT-Release:** Trending Tags sofort aktivieren
- **Nach 10 Videos:** Erste belastbare Daten, welche Tags Traffic bringen

### Was prüfen?
1. Welche **Suchbegriffe** bringen Zuschauer? → Diese Tags verstärken
2. Welche Tags bringen **keinen Traffic**? → Austauschen gegen neue
3. Gibt es **neue Autocomplete-Vorschläge**? → YouTube-Suche erneut checken
4. Wie performen die **Long-Tail Keywords**? → Erfolgreiche in mehr Videos nutzen

### Wie updaten?
1. YouTube Studio → Video auswählen → Details → Tags bearbeiten
2. Schwache Tags entfernen, neue einsetzen
3. Beschreibungs-Hashtags anpassen wenn nötig
4. Änderungen in dieser Datei dokumentieren
