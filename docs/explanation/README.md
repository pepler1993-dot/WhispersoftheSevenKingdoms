# Explanation

Explanation-Dokumente beantworten das **Warum**.

Hier geht es nicht primär um Bedienung oder Lookup, sondern um:
- Architekturentscheidungen
- Tradeoffs
- strategische Richtungen
- Gründe für aktuelle Prioritäten
